Документация программы YonnCalc

Общая информация

Программа YonnCalc представляет собой Desktop-программу, которая позволяет выполнять различные вычисления и построение графиков функций.

Ввод выражения

В программе YonnCalc вы можете вводить арифметические выражения в инфиксной нотации, используя следующие операторы и функции:

  • Скобки: ( )
  • Сложение: +
  • Вычитание: -
  • Умножение: *
  • Деление: /
  • Возведение в степень: ^
  • Остаток от деления: mod
  • Унарный плюс: +a
  • Унарный минус: -a
  • Функция косинуса: cos(X)
  • Функция синуса: sin(X)
  • Функция тангенса: tan(X)
  • Функция арккосинуса: acos(X)
  • Функция арксинуса: asin(X)
  • Функция арктангенса: atan(X)
  • Функция квадратного корня: sqrt(X)
  • Функция натурального логарифма: ln(X)
  • Функция десятичного логарифма: log(X)

Обратите внимание, что вы можете использовать переменную X в выражениях для вычисления функций.

Доступен ввод нажатием на кнопки приложения, а также ввод с клавиатуры.

Вычисление выражения

Чтобы вычислить введенное выражение, введите его полностью и нажмите на символ "=". Программа Калькулятор выполнит вычисления и отобразит результат.

Построение графика функции

В программе Калькулятор вы также можете построить график функции, заданной в виде выражения в инфиксной нотации с переменной X. Для построения графика необходимо указать область определения и область значений функции. График будет отображен с координатными осями, масштабом и сеткой с адаптивным шагом.

Ограничения

При использовании программы YonnCalc следует учитывать следующие ограничения:

  • Область определения и область значений функций ограничиваются числами от -1000000 до 1000000.
  • Точность дробной части составляет минимум 7 знаков после запятой.
  • Длина вводимого выражения ограничена 255 символами.
  • Масштаб графика не может быть изменен пользователем.

Пример использования

Вот примеры выражений, которые можно вводить в программу Калькулятор:

  • Вычисление: (2 + 3) * 4
  • Вычисление с использованием переменной: sin(X) + cos(X)
  • Построение графика: X^2, отображаемая область определения: [-10, 10], область значений: [-100, 100]
