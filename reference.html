YonnCalc Documentation

General Information

YonnCalc is a desktop application that allows you to perform various calculations and plot graphs of functions.

Expression Input

In YonnCalc, you can enter arithmetic expressions in infix notation using the following operators and functions:

  • Brackets: ( )
  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Exponentiation: ^
  • Modulus: mod
  • Unary plus: +a
  • Unary minus: -a
  • Cosine function: cos(X)
  • Sine function: sin(X)
  • Tangent function: tan(X)
  • Arccosine function: acos(X)
  • Arcsine function: asin(X)
  • Arctangent function: atan(X)
  • Square root function: sqrt(X)
  • Natural logarithm function: ln(X)
  • Decimal logarithm function: log(X)

Note that you can use the variable X in expressions to compute functions.

Input can be done by clicking the buttons in the application or by using the keyboard.

Expression Evaluation

To evaluate the entered expression, input it completely and press the "=" symbol. YonnCalc will perform the calculations and display the result.

Function Graphing

In YonnCalc, you can also plot the graph of a function defined by an expression in infix notation with the variable X. To plot a graph, you need to specify the domain and range of the function. The graph will be displayed with coordinate axes, a scale, and a grid with adaptive spacing.

Limitations

When using YonnCalc, please consider the following limitations:

  • The domain and range of functions are limited to numbers from -1000000 to 1000000.
  • The decimal precision is at least 7 digits after the decimal point.
  • The length of the input expression is limited to 255 characters.
  • The scale of the graph cannot be changed by the user.

Usage Examples

Here are some example expressions that can be entered into YonnCalc:

  • Calculation: (2 + 3) * 4
  • Calculation with variable: sin(X) + cos(X)
  • Graphing: X^2, domain: [-10, 10], range: [-100, 100]
